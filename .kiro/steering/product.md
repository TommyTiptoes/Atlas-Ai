# Visual AI Personal Assistant - Product Overview

## Product Summary

The Visual AI Personal Assistant is a Windows desktop application that combines voice control, 3D avatar visualization, and AI-powered task automation. The system emphasizes transparency and user control through an "explain-before-execute" approach.

## Core Features

- **Voice-Controlled AI Assistant**: Natural language commands with multiple AI provider support (Claude, OpenAI, Azure OpenAI, Google AI, Local models)
- **3D Avatar Interface**: Unity-based visual assistant with emotions, animations, and lip-sync
- **Transparent Operation**: Explain-before-execute workflow with user approval gates
- **Action Timeline**: Complete history with undo/rollback capabilities
- **Skill Learning**: "Watch Me" mode to learn from user demonstrations
- **Multi-Role System**: Different AI personas (Assistant, Developer, Analyst, etc.)
- **Memory Management**: Transparent, editable memory with privacy controls
- **Privacy Mode**: Local-only operation with no external API calls

## Target Platform

- **Primary**: Windows 10/11 desktop application
- **Runtime**: .NET 8.0 with WPF for UI
- **Avatar Engine**: Unity 3D for 3D avatar rendering
- **Architecture**: Hybrid WPF + Unity integration

## Key Differentiators

1. **Transparency**: Users can see and control everything the AI remembers and plans to do
2. **Learning**: AI learns from watching user demonstrations rather than just following commands
3. **Visual Feedback**: 3D avatar provides engaging visual interaction
4. **Privacy-First**: Multiple privacy levels including complete offline mode
5. **Rollback System**: Complete action history with undo capabilities
